

### Simple alert

```js
        A simple primary alert—check it out!
    </Alert>

    <Alert color='secondary'>

    </Alert>

    <Alert color='success'>

    </Alert>

    <Alert color='danger'>

 ```
### Dismissable alerts

```js
<div>
    <Alert dismissable color='warning'>
        Has cibo omittam percipitur an, no duo erat placerat oporteat.
    </Alert>
</div>
```



### Alert with a body

```js

<div>
    <Alert color='primary'>
        <Alert.Body>
            Vix an perfecto consequuntur, altera nostrum intellegam ex mel in.
        </Alert.Body>
    </Alert>
</div>

```


### Alert with header, body and separator

```js

    <Alert color='primary'>
        <Alert.Header>
            Header
        </Alert.Header>
        <Alert.Separator>
        </Alert.Separator>
        <Alert.Body>
            Body
        </Alert.Body>
    </Alert>

 ```
